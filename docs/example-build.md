# `omg build` example

The [random](https://github.com/microservice/random) microservice is used for this example.

## Vanila

### Input
```
omg build
```

### Output
```
✔ Built Docker image with name: omg/microservice/random
```

## `-t` flag

### Input
```
omg build -t rand
```

### Output
```
✔ Built Docker image with name: rand
```
